# E-commerce-Back-end

## Description

 Back-end build for an E-commerce website using api routes, a database with mysql and express as our server.
 

## Technologies Used

 * mysql
 * Node.js
 * Express.js
 * Sequelize
 * Javascript
 * insomnia

## Usage 
* npm start
